This project is supposed to be website made for better
memorising experience with memory cards (on one side question,
on the other: answer).

Plans:
• Realize cards storage.
• Keep users' passwords encrypted.
• Create stable working minimalistic web version.
• Add some featerus to the website, such as:
- animationed cards with possibility to move them from one category to another;
- standard public cards storage for usual for students topics;
- add possibility to attach images.
